Public gateway for SPEI (instant interbank transfers) integration. Receive and send via SPEI, query balance and transactions, receive signed webhooks.

## Environments

| Environment | URL |
|---|---|
| Sandbox | `https://sandbox.mx.ntxpay.com` |
| Production | Provided at onboarding |

## Authentication

1. Your credentials (`clientId` + `clientSecret`) are provided during onboarding.
2. `POST /api/auth/token` with X.509 certificate in the `X-SSL-Client-Cert` header + credentials returns a JWT (10 min validity).

The JWT goes in `Authorization: Bearer <token>` for the other endpoints.

## Monetary values

MXN centavos, integers. `50000` = $500.00 MXN.

## Dates

ISO 8601 UTC: `2026-05-12T14:31:05.000Z`.

## HTTP Codes

| Code | Meaning |
|---|---|
| `200` / `201` | Success |
| `400` | Invalid data or insufficient balance |
| `401` | Invalid token or certificate |
| `404` | Resource not found |
| `429` | Rate limit |
| `502` | Service temporarily unavailable |
