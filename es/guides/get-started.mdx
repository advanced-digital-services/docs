## Visión General

La API NTX Pay México permite que tu empresa realice operaciones de pago, concilie transacciones y reciba notificaciones por webhooks — todo de forma segura y escalable. La autenticación es vía **certificado X.509 (mTLS) + OAuth 2.0**, y devuelve un **JWT** de corta duración para cada solicitud.

## Paso a Paso en el Sandbox

<Steps>
  <Step title="Solicita una cuenta sandbox">
    Contacta a tu gerente de cuenta o escribe a `contact@ntxpay.com` solicitando una cuenta sandbox.
  </Step>
  <Step title="Recibe tus credenciales">
    Recibirás un **certificado** y las **credenciales OAuth** (`clientId` + `clientSecret`) asociadas a la cuenta sandbox.
  </Step>
  <Step title="Registra la URL del webhook">
    Registra la URL de tu webhook vía `POST /api/webhooks-config` para recibir eventos simulados.

    ```bash
    curl -X POST https://sandbox.mx.ntxpay.com/api/webhooks-config \
      -H "Authorization: Bearer $TOKEN" \
      -H "Content-Type: application/json" \
      -d '{
        "url": "https://meu-servidor.com/webhooks/ntxpay",
        "events": ["cash_in"]
      }'
    ```
  </Step>
  <Step title="Autentícate y simula escenarios">
    Autentícate, haz llamadas a la API y usa el header `X-Sandbox-Scenario` para simular diferentes resultados (confirmación inmediata, falla, expiración, etc.).

    ```bash
    curl -X POST https://sandbox.mx.ntxpay.com/api/spei/cash-in \
      -H "Authorization: Bearer $TOKEN" \
      -H "X-Sandbox-Scenario: rejected" \
      -H "Content-Type: application/json" \
      -d '{
        "amountCentavos": 50000,
        "externalId": "order-001",
        "customerName": "Juan Perez",
        "customerEmail": "juan@example.com"
      }'
    ```
  </Step>
</Steps>
